# PlanH V2

这是 [planH_BOOT](https://github.com/kanetah/planH_BOOT) 项目的彻底重构版本，目标是完成一个作业提交平台。  
采用前后端分离的方式进行编写，其中，后端使用 `Kotlin` + `Spring Boot` 编写，前端则主要用 `ES6` + `React` 编写。

目前正在施工中...

### 关于前端
暂定借助于蚂蚁家的 ant-design 进行开发
